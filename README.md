# Creator Space Frontend 

This project is a React application demonstrating client-side routing using React Router.

---

## 📌 Features

* Navigation using `NavLink`
* Multiple pages:

  * Dashboard
  * Profile
  * Settings
* No page reload routing (SPA behavior)

---

## 🛠️ Tech Stack

* React.js
* React Router DOM
* JavaScript
* CSS

---
